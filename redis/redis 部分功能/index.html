慢查询 是指 在 执行命令阶段

客户端超时不一定是慢查询，但慢查询可能会导致客户端超时

pipeline

redis 一条命令的时间往往是 微秒级别的

一次 pipeline 时间 = 1 次网络时间 + n 次命令时间

注意点

  1. 每个 pipeline 携带的数据量
  2. pipeline 每次只作用在 一个 redis 节点上

发布订阅,消息队列

订阅者无法获取 频道 订阅前的消息

  1. publish 发布
  2. subscribe 订阅
  3. unsubscribe 取消订阅
